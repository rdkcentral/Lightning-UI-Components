# Themes

UI Components support creating a global theme.

## Source

https://github.comcast.com/Lightning/lightning-ui/packages/@lightning/ui-core/src/components/theme.stories/theme.stories.js

```js
import lng from '@lightningjs/core';
import createTheme from '@lightning/ui/Styles';
lng.uiTheme = createTheme({

});
```

## Xfinity

Default theme for Xfinity-branded components.

```js
import { getXfinityTheme } from '@lightning/ui/Styles';
const theme = getXfinityTheme();
```
