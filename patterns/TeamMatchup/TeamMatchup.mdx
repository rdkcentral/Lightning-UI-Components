# TeamMatchup

Pattern for Data Item that displays matchup information.

## Usage

`TeamMatchup` can display entity specific logo/icon and a background gradient that reflects the colors on both enitities.

```js
import { TeamMatchup } from '@lightning/ui/patterns';

Matchup: {
  type: TeamMatchup,
  subtitle: '1st - 1:15',
  title: '24 - 13',
  removeBackground: false,
  metadata: 'Eagles vs Giants',
  items: {
    left: {
      name: 'Eagles',
      color: 'rgba(8,111,107,1)',
      src: path/to/logo
    },
    right: {
      name: 'Giants',
      color: 'rgba(11,81,161,1)',
      src: path/to/logo
    }
  }
}
```

### Properties

name|type|required|default|description
--|--|--|--|--
title|text|false|-|title text
subtitle|text|false|-|subtitle text
removeBackground|boolean|false|-|flag to remove background
items|object|true|-|details of left/right entities to be displayed
metadata|string|true|-|metadata to be displayed on focus

### Items

name|type|required|default|description
--|--|--|--|--
name|text|true|-|name of entity for get announce
color|number|true|-|entity color
src|path|true|-|path of entity image/icon
