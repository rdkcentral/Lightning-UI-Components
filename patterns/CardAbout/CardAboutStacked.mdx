# CardAboutStacked

This component is used to display two titles with content below to create a stacked effect.

## Usage

`CardAboutStacked` takes in a title and a titleBottom to create a stacked effect. contentBottom is the inline content that can be passed in to go below the title, and descriptionBottom can be passed in to go below titleBottom.
`CardAboutStacked` extends [Card](?path=/docs/patterns-card--base)

```js
import { CardAboutStacked } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      CardAboutStacked: {
        type: CardAboutStacked,
        title: 'Audio',
        titleBottom: 'Title for the bottom part of the card'
      }
    };
  }
}
```

## API

### Properties

| name              | type   | required | default | description                              |
| ----------------- | ------ | -------- | ------- | ---------------------------------------- |
| description       | object | false    | -       | inline content in the middle of the card |
| titleBottom       | string | false    | -       | title on the bottom half of the card     |
| descriptionBottom | string | false    | -       | bottom description                       |

### Style Properties

| name                      | type   | description                      |
| ------------------------- | ------ | -------------------------------- |
| descriptionTextProperties | object | text style of the description    |
| paddingVertical           | string | padding vertically between lines |
