# ContentTray

Allows for row of Lightning components to be used as "Tabs" to display its child items as below when focused. Similar to [TabBar](?path=/story/elements-tabbar--basic).

## Usage

The `ContentTray` pattern expects an array of `items` ("Tabs") to be passed to it. Each of those `items` ("Tabs") expects an array of `items` ("Results") which will display when its parent `item` ("Tab") gains focus.

```js
import { ContentTray } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ContentTray: {
        itemSpacing: 40,
        type: ContentTray,
        items: [
          {
            type: Tile,
            title: 'Tile 1',
            w: 100,
            h: 100,
            radius: 50,
            imgRadius: 50,
            src: 'https://url.for/tileImage',
            items: [
              {
                type: CardLaunchpad,
                src: `https:/url.for/cardImage`,
                shouldAnimate: false,
                focused: true,
                title: 'Program Title',
                description: 'Short description'
              },
              {
                type: CardLaunchpad,
                src: `https:/url.for/cardImage`,
                shouldAnimate: false,
                focused: true,
                title: 'Program Title',
                description: 'Short description'
              },
              {
                type: CardLaunchpad,
                src: `https:/url.for/cardImage`,
                shouldAnimate: false,
                focused: true,
                title: 'Program Title',
                description: 'Short description'
              }
            ]
          }
        ]
      }
    };
  }
}
```

### Properties

| name        | type    | required | default   | description                                                                                                                                                         |
| ----------- | ------- | -------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| collapse    | boolean | false    | undefined | will collapse results when ContentTray does not have focus                                                                                                          |
| items       | array   | false    | {}        | an array of objects that represent components in the `Tabs` row, which themselves can have another level of `items` which represent components in the `Results` row |
| itemSpacing | number  | false    | 0         | horizontal gutter between items                                                                                                                                     |
| reset       | boolean | false    | undefined | will reset Tabs / Results Rows when loses focus                                                                                                                     |
