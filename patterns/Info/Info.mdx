# Info

Generic pattern for Data Item specific to series/episode.

```js
import { Info } from '@lightning/ui/patterns';

Info: {
  type: Info,
  title: 'Series Title',
  subtitle: 'Season 5 - Info 2',
  description: 'This is a description of episode 2 from Season 5 of Series Title'
}
```

### Properties

name|type|required|default|description
--|--|--|--|--
title|text|false|-|text to display in title
subtitle|text|false|-|text to display in subtitle
description|text|false|-|text to display in description
announce|text|false|title + subtitle + description|override default announce
