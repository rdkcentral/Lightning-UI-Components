# SportStats

Pattern for Data Item that displays game stats or team stats

## Usage

If displaying multiple entities, the items array should include the name, entity color and value of the statistic. For single entities, only the value is required.

```js
import { SportStats } from '@lightning/ui/patterns';

Stats: {
  type: SportStats,
  stats: [
    {
      label: 'Total yards',
      items: [
        {
          name: 'Eagles',
          color: 'rgba(8,111,107,1)',
          value: 252
        },
        {
          name: 'Giants',
          color: 'rgb(11,81,161)',
          value: 243
        }
      ]
    },
    {
      label: 'Turnovers',
      items: [
        {
          name: 'Eagles',
          color: 'rgb(8,111,107)',
          value: 2
        },
        {
          name: 'Giants',
          color: '4278931873',
          value: 2
        }
      ]
    },
    {
      label: 'First downs',
      items: [
        {
          name: 'Eagles',
          color: '4278742891',
          value: 12
        },
        {
          name: 'Giants',
          color: 'rgba(11,81,161,1)',
          value: 15
        }
      ]
    },
    {
      label: 'Red zone %',
      percentage: true,
      items: [
        {
          name: 'Eagles',
          color: 'rgb(8,111,107)',
          value: 100
        },
        {
          name: 'Giants',
          color: '4278931873',
          value: 66
        }
      ]
    }
]
}
```

If displaying statistics for a single entity, the items array should only contain a single entitiy

```js
import { SportStats } from '@lightning/ui/patterns';

Stats: {
  type: SportStats,
  stats: [
    {
      label: 'Total yards',
      items: [{
        name: 'Eagles',
        color: '4278742891',
        value: 252
      }]
    },
    {
      label: 'Turnovers',
      items: [{
        name: 'Eagles',
        color: '4278742891',
        value: 2
      }]
    },
    {
      label: 'First downs',
      items: [{
        name: 'Eagles',
        color: '4278742891',
        value: 12
      }]
    },
    {
      label: 'Red zone %',
      percentage: true,
      items: [{
        name: 'Eagles',

```

### Properties

name|type|required|default|description
--|--|--|--|--
stats|array\object|true|-|statistics array

### Stats

name|type|required|default|description
--|--|--|--|--
label|text|true|-|stat label
percentage|boolean|false|-|display statistic as a decimal
items|array\object|true|-|entity details and stat values

### Items

name|type|required|default|description
--|--|--|--|--
name|text|true|-|name of entity
color|number|true|-|entity color
value|string|true|-|value of statistic
