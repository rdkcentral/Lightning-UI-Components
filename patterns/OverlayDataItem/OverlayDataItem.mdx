# OverlayDataItem

The OverlayDataItem extends [DataItem](?path=/story/patterns-dataitem--basic) to allow a gradient background or an image overlay, accepting all properties of `DataItem`.

```js
import { OverlayDataItem } from '@lightning/ui/patterns';

class Example extends lng.Component {
  static _template() {
    return {
      OverlayDataItem: {
        h: 140,
        w: 410,
        type: OverlayDataItem,
        backgroundColors: {
          colorLeft: 'rgba(8,111,107,1)',
          colorRight: 'rgba(11,81,161,1)'
        }
      }
    };
  }
};
```

## API

### Properties

name|type|required|default|description
--|--|--|--|--
overlayColor|rbg\|rgba\|number\|hex|false|-|overlay color for background
overlayImage|path|false|-|path to overlay image src
backgroundColors|object|false|-|left/right colors for background gradient

### backgroundColors

name|type|required|default|description
--|--|--|--|--
colorLeft|rbg\|rgba\|number\|hex|true|-|left gradient color
colorRight|rbg\|rgba\|number\|hex|true|-|right gradient color
