# withBadgeProgress

Adds a configurable progress bar and badge to the element. The element should have a height at least equal to the combined height of the badge and progress bar, and a width of at least 2x the badge width.

## Usage

To leverage the mixin, wrap an existing component with it and provide the expected properties `badge`, `badgeLocation`, and/or `progress`. Define the wrapped component as you normally would.

__Note__: because this mixin relies on the wrapped component's dimensions, there can be positioning issues if these properties aren't available quickly enough. If you notice this bug, try setting the `w` and `h` of the wrapped component in it's `_init` method.


```js
import { withBadgeProgress } from '@lightning/ui/patterns';

class Example extends lng.Component {
  static _template() {
    return {
      withBadgeProgress: {
        type: withBadgeProgress(Tile),
        Tile: {
          // ...Tile properties
        },
        badge: '1:45',
        badgeLocation: 'lowerLeft',
        progress: 0.7
      }
    };
  }
};
```

## API

### Properties

name|type|required|default|description
--|--|--|--|--
badge|string|false|undefined|the content of the badge
badgeLocation|string|false|'upperLeft'|position the badge will appear over the wrapped component
progress|number|false|0|percentage of the current progress from 0 to 1
