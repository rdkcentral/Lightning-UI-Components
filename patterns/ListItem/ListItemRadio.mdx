# ListItemRadio

A [`ListItem`](?path=/docs/listitem--basic) with dedicated [`Radio`](?path=/docs/toggle--basic) functionality

## Usage

`ListItemRadio` can display a `title` and `subtitle`. The radio itself sits on the right side:

```js
import { ListItemRadio } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItemRadio: {
        type: ListItemRadio,
      }
    };
  }
};
```

`ListItemRadio` will always toggle on <kbd>Enter</kbd>. This can not be overriden.

To get the current checked state, use the `isChecked()` method.

```js
if (ListItemRadio.isChecked()) {
  // do something
}
```

## API

### Properties

name|type|readonly|description
--|--|--|--
backgroundType|'fill'\|'float'\|'ghost'|false|color of `ListItem` container.
checked|boolean|false|sets initial `checked` value of Radio. Use `toggle()` to change value after init
icon|string|false|path to icon image
size|'small'\|'medium'\|'large'\|number|false|width of the `ListItem` container
subtitle|string|false|item metadata info goes here
title|string|false|main item info goes here

### Methods

#### isChecked():boolean

returns the checked value (boolean) for the Radio

#### toggle():boolean

Toggles the `checked` state and returns the new state.
