# ListItemImage

A [`ListItem`](?path=/docs/listitem--basic) with a [`Tile`](?path=/docs/tile--basic) image

## Usage

`ListItemImage` can display a `title`, `subtitle`, and `image`. The expected image size is 56x56.

```js
import { ListItemImage } from '@lightning/lightning-ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItem: {
        type: ListItem,
        title: 'List Item',
        subtitle: 'List item metadata',
        image: 'path/to/image.png'
      }
    };
  }
};
```

## API

### Properties

name|type|readonly|description
--|--|--|--
backgroundType|'fill'\|'float'\|'ghost'|false|color of `ListItem` container
icon|string|false|path to icon image
size|'small'\|'large'\|number|false|width of the `ListItem` container
image|string|false|path to image
subtitle|string|false|item metadata info goes here
title|string|false|main item info goes here
imageSize|number|false|applied to height and width of the image which shoule be square and defaults to 56

### Methods

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event

#### set texture(newTexture): void

Allows the image to be overriden with a custom texture component
Example:
```
let newTexture = {
  type: lng.textures.ImageTexture,
  resizeMode: { type: 'cover', w: 56, h: 56 },
  src: 'pathToImage.jpg'
};
this.tag('ListItemImage').texture = newTexture;
```
