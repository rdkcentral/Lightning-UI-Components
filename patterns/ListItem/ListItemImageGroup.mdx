# ListItemImageGroup

A [`DataItem`](?path=/docs/patterns-dataitem--basic) containing a list of [`ListItemImage`](?path=/docs/patterns-listitemimage--basic) that does not focus on individual items

## Usage

Just set a src property to a listItem to load an image.

```js
import { ListItemImageGroup } from '@lightning/ui/patterns';

ListItemImageGroup: {
  type: ListItemImageGroup,
  listItems: [
    {
      title: 'Jessia Findlay',
      subtitle: 'Lenina Crowne',
      src: 'path/to/image'
    },
    {
      title: 'Alden Ehrenreich',
      subtitle: 'John the Savage',
      src: 'path/to/image'
    },
    {
      title: 'Harry Lloyd',
      subtitle: 'Bernard Marx',
      src: 'path/to/image'
    }
  ]
}
```
