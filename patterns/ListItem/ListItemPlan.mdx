# ListItemPlan

A [`ListItem`](?path=/docs/listitem--basic) with a 3rd line text content and [`Radio`](?path=/docs/toggle--basic) functionality.

## Usage

`ListItemPlan` can display a `title`, `subtitle`, and `content`. The radio itself sits on the right side:

```js
import { ListItemPlan } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItemPlan: {
        type: ListItemPlan,
        title: 'ListItemPlan Title',
        subtitile: 'ListItemPlan subtitle',
        content: 'ListItemPlan content'
      }
    };
  }
}
```

`ListItemPlan` will always toggle on <kbd>Enter</kbd>. This can not be overriden.

To get the current checked state, use the `isChecked()` method.

```js
if (ListItemPlan.isChecked()) {
  // do something
}
```

## API

### Properties

| name           | type                                         | readonly | description                                                                      |
| -------------- | -------------------------------------------- | -------- | -------------------------------------------------------------------------------- |
| title          | string                                       | false    | the 1st line, main item info goes here                                                         |
| subtitle       | string                                       | false    | the 2nd line, item metadata info goes here                                                     |
| content        | string                                       | false    | the 3rd line, the details of item, can be undefined or set to empty string so this line won't show|
| checked        | boolean                                      | false    | sets initial `checked` value of Radio. Use `toggle()` to change value after init |
| backgroundType | 'fill'\|'float'\|'ghost'                     | false    | color of `ListItem` container.                                                   |
| checked        | boolean                                      | false    | sets initial `checked` value of Radio. Use `toggle()` to change value after init |
| size           | 'xsmall'\|'small'\|'medium'\|'large'\|number | false    | width of the `ListItem` container, only works when w is undefined or 0.<br /><br />**RECOMMEND** setting prop `w` instead since this componenet might need customized width                                         |


### Methods

#### isChecked():boolean

returns the checked value (boolean) for the Radio

#### toggle():boolean

Toggles the `checked` state and returns the new state.
