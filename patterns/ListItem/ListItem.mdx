# ListItem

Component to be displayed in a list

## Usage

`ListItem` can display a `title`, `subtitle`, and `icon`:

```js
import { ListItem } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItem: {
        type: ListItem,
        title: 'List Item',
        subtitle: 'List item metadata',
        icon: 'path/to/icon.png'
      }
    };
  }
};
```

### Background

For a transparent background, set `backgroundType: 'float'`. The default value is `fill`.

### Sizing

There are three sizing options, `small`, `large`, and any positive integer. The default value is `large`. These options only affect width, the height is fixed at 88px.

### Focus

When focused, the background/foreground colors are inverted.

### Icons

Adding icons to a `ListItem` is optional, and they can be added one of two ways:

**Pass a url path**

```js
{
  type: ListItem,
  icon: 'path/to/icon.png'
}
```

**Pass a list of paths**

```js
{
  type: ListItem,
  icon: ['path/to/icon.png', 'path/to/other/icon.png']
}
```

If multiple icons are passed, they will be laid out horizontally in order.

Any common image extension can be passed as `icon`, or an inline `SVG`. The expected icon dimension is 40x40.

## API

### Properties

name|type|readonly|description
--|--|--|--
backgroundType|'fill'\|'float'\|'ghost'|false|color of `ListItem` container
icon|string\|string[]|false|path to icon image (or list of paths). See <a href="?path=/docs/icon--basic">Icon</a>
size|'small'\|'large'\|number|false|width of the `ListItem` container
subtitle|string|false|item metadata info goes here
title|string|false|main item info goes here
collapse|bool|false|set to true to hide subtitle when ListItem is not in focus
announce|string|true|label or title|announcement text for screen readers
collapseIcon|boolean|false|hide the list item icon(s) on focus
### Methods

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event
