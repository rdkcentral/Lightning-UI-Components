# Lists

Collection of Lightning UI List patterns.

## Usage

* [ListItem](?path=/docs/patterns-listitem--basic)
* [ListItemRadio](?path=/docs/patterns-listitemradio--basic)
* [ListItemToggle](?path=/docs/patterns-listitemtoggle--basic)
* [ListItemSlider](?path=/docs/patterns-listitemslider--basic)
* [ListItemPicker](?path=/docs/patterns-listitempicker--basic)
* [ListItemImage](?path=/docs/patterns-listitemimage--basic)

List Items are used in groups to support configuring the app experience. List Items can include many variations of text, controls, icons & images.

Please reference the [UX documentation](https://etwiki.sys.comcast.net/display/XDSS/List+items) for more information.
