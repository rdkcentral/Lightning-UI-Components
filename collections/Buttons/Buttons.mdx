# Buttons

Collection of Lightning UI Button components.

## Usage

* [ActionButton](?path=/docs/elements-actionbutton--base)
* [ActionButtonLarge](?path=/docs/elements-actionbuttonlarge--base)
* [ActionButtonMedium](?path=/docs/elements-actionbuttonmedium--base)
* [ActionButtonSmall](?path=/docs/elements-actionbuttonsmall--base)
* [ActionButtonIconLarge](?path=/docs/elements-actionbuttoniconlarge--base)
* [ActionButtonIconMedium](?path=/docs/elements-actionbuttoniconmedium--base)
* [ActionButtonIconSmall](?path=/docs/elements-actionbuttoniconsmall--base)

There are no prescribed uses for each type of button, instead you should choose which type best fits your design. 

Please reference the [UX documentation](https://etwiki.sys.comcast.net/display/XDSS/Action+buttons) for more information.
