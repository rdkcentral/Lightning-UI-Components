# withMetadata

The `withMetadata` mixin can be wrapped around any Tile component to position Metadata underneath it. This mixin is not designed for any non-Tile or non-Tile variant components. For example, TileIcon is valid as it extends Tile, but you should not try to use DataItem.

## Usage

Pass in the Tile type to the `withMetadata` mixin and a Metadata component to display with the Tile.

```js
import { withMetadata } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      TileWithMetadata: {
        type: withMetadata(Tile),
        w: 320,
        h: 180,
        persistentMetadata: true,
        Metadata: {
          type: MetadataCard,
          firstLine: 'metadata text here'
        }
      }
    };
  }
};
```

### Properties

name|type|required|default|description
--|--|--|--|--
Metadata|lng.Component|false|-|the component that will be added underneath the [Tile](?path=/docs/tile--basic), such as [MetadataCard](?path=/docs/metadatacard--basic) on either side or [MetadataTile](?path=/docs/metadatatile--basic) on either side
persistentMetadata|boolean|false|false|determines whether the metadata should only be visible on focus or all the tile
progress|number|false|-|sets the progress percentage of the [ProgressBar](?path=/docs/progressbar--basic)
badge|number|false|-|sets the title text for [Badge](?path=/docs/badge--basic)
paddingTop|number|false|8|sets the amount of space between the bottom of the [Tile](?path=/docs/tile--basic) and top of the Metadata
paddingSide|number|false|16|sets the amount of space the Metadata should be inset from the [Tile](?path=/docs/tile--basic) on either side
badgePadding|number|false|24|sets the amount of space the [Badge](?path=/docs/badge--basic) on the top and left should be inset from the [Tile](?path=/docs/tile--basic) on the top and left
progressBarPadding|number|false|24|sets the amount of space the [ProgressBar](?path=/docs/progressbar--basic) should be inset from the [Tile](?path=/docs/tile--basic) on the sides and bottom
