# CardTitle

Component shows title and description within a card.

## Source

https://github.comcast.com/Lightning/lightning-ui/blob/develop/packages/@lightning/ui/src/components/Card/CardTitle.js

## Usage

`CardTitle` can display both a title and a description if a title and description are passed in.
`CardTitle` extends [Card](?path=/docs/patterns-card--base)

```js
import { CardTitle } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Card: {
        type: CardTitle,
        title: 'CardTitle',
        description: 'Description for the Card Title',
        details: 'Details'
      }
    };
  }
}
```

## API

### Properties

| name        | type   | required | default   | description |
| ----------- | ------ | -------- | --------- | ----------- |
| description | string | false    | undefined | description |
| details     | string | false    | undefined | details     |

### Style Properties

| name                      | type            | description                        |
| ------------------------- | --------------- | ---------------------------------- |
| descriptionTextProperties | string \|object | text style of the description text |
| detailsTextProperties     | string \|object | text style of the details text     |
