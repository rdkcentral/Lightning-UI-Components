# TextBox

General purpose TextBox component

## Source

https://github.comcast.com/Lightning/lightning-ui/blob/develop/packages/@lightning/ui-core/src/components/TextBox/TextBox.js

## Usage

Set a content property with your text and specify a property style as a string.

```js
import { TextBox } from '@lightning/ui';

TextBox: {
  type: TextBox,
  content: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit.',
  style: 'headline1',
  w: 600,
}
```

### Properties

| name           | type          | required | default                | description                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| -------------- | ------------- | -------- | ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| content        | string        | true     | -                      | Text to be displayed in element                                                                                                                                                                                                                                                                                                                                                                                                                     |
| maxLines       | number        | false    | 0                      | Maximum number of lines to display before truncation.                                                                                                                                                                                                                                                                                                                                                                                               |
| maxLinesSuffix | string        | false    | '...'                  | String to be displayed at the end of the line when truncation is used                                                                                                                                                                                                                                                                                                                                                                               |
| textAlign      | string        | false    | left                   | Avalilable values 'left', 'right', or 'center'.                                                                                                                                                                                                                                                                                                                                                                                                     |
| textColor      | hex           | false    | #ffffff                | Desired text color.                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| textStyle      | string/object | false    | theme.typography.body1 | Style of text to be displayed. This value will take precedent over the `textStyle` style property. Available values display1, display2, headline1, headline2, headline3, body1, body2, body3, button1, button2, callout1, caption1. Alternatively an object can be passed in containing any [properties that the Lightning text texture supports](https://lightningjs.io/docs/#/lightning-core-reference/RenderEngine/Textures/Text?id=properties). |
| verticalAlign  | string        | false    | middle                 | When lineHeight is larger than fontSize, specifies how text should align vertically inside each line. Available values 'top', 'middle', or 'bottom'.                                                                                                                                                                                                                                                                                                |
| wordWrap       | boolean       | false    | true                   | Allows long sentences to be broken into new lines.                                                                                                                                                                                                                                                                                                                                                                                                  |
| wordWrapWidth  | number        | false    | 0                      | Width set to start wrapping to next line.                                                                                                                                                                                                                                                                                                                                                                                                           |

TextBox also supports rendering as an InlineContent component. See [InlineContent documentation](/?path=/docs/layout-inlinecontent--basic) for table of supported properties.

### Style Properties

| name           | type             | description                                                                                                                                                                                                                                             |
| -------------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| maxLinesSuffix | string           | String to be displayed at the end of the line when truncation is used.                                                                                                                                                                                  |
| offsetX        | number           | Used to ensure text is positioned correctly                                                                                                                                                                                                             |
| offsetY        | number           | Used to ensure text is positioned correctly                                                                                                                                                                                                             |
| textStyle      | string \| object | Default style of text to be displayed. Object should come from the current theme or contain any properties that the [Lightning text texture supports](https://lightningjs.io/docs/#/lightning-core-reference/RenderEngine/Textures/Text?id=properties). |
| typography     | object           | The typography object set on the current theme. Pass the key as string to `textStyle` to use the associated text style on the typography object.                                                                                                        |
