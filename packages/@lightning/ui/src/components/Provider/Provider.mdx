# Provider

The `Provider` serves to condense a list of providers to show only the number you specify, and add a counter to represent any that remain.

## Source

https://github.comcast.com/Lightning/lightning-ui/blob/develop/packages/@lightning/ui/src/components/Provider/Provider.js

## Usage

The `Provider` component only needs an array of image paths to populate the list of providers. However, you can supply an `Icon` component, and the `Provider` will resize it to fit. This is useful, for example, if you want to include a 2:1 provider logo amongst a series of 1:1s.

```js
import { Provider } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Provider: {
        type: Provider,
        visibleCount: args.visibleCount,
        providers: [
          xfinityProviderLogoSquare,
          {
            type: Icon,
            w: 96,
            h: 48,
            icon: xfinityLogo
          },
          ...Array.apply(null, { length: 10 }).map(
            () => xfinityProviderLogoSquare
          )
        ],
        counterText: '...',
        disableRadius: true
      }
    };
  }
}
```

## API

### Properties

| name            | type    | required | default | description                                                                      |
| --------------- | ------- | -------- | ------- | -------------------------------------------------------------------------------- |
| disableRadius   | boolean | false    | -       | if true, disables the application of the radius style property to Icon providers |
| counterText     | string  | false    | -       | text to override counter's default '+Number' functionality                       |
| providers       | array   | true     | -       | list of provider images                                                          |
| providersHidden | number  | readonly | -       | the number of providers that will be hidden by the counter                       |
| visibleCount    | number  | true     | -       | the number of providers to show before adding a counter                          |

### Style Properties

| name                   | type          | description                                                     |
| ---------------------- | ------------- | --------------------------------------------------------------- |
| counterBackgroundColor | string        | background color of counter                                     |
| counterTextStyle       | object        | text style for counter                                          |
| itemSize               | number        | height and width to apply to icons and counter                  |
| itemSpacing            | number        | space between providers                                         |
| radius                 | number\|array | radius of counter (and Icons if `applyRadius` property is true) |
