# Notification

Component that animates in and out of view to display an icon, title, description, and action area.

## Source

https://github.comcast.com/Lightning/lightning-ui/blob/develop/packages/@lightning/ui/src/components/Notification/Notification.js

## Usage

`Notification` accepts an icon, title, description, and action area. If an action area is provided, the notification will increase its height to display that section.
Title and description text wrap, action area text truncates. The enter and dismiss animations consist of 3 steps which expand or collapse the Notification to display all
provided content.

```js
import { Notification } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Notification: {
        type: Notification,
        icon: 'path/to/icon',
        title: 'Title',
        description: 'This is the description.'
        actionArea: 'Action area'
      }
    };
  }
}
```

### Animations Steps

#### enter:

1. Expand and alpha in to display only the icon
2. Expand the height and width from top right corner of the Notification to display the icon, title, and description
3. Expand the height downwards to display the action area

#### dismiss:

3. Collapse the height upwards to hide the action area
4. Collapse the height and width from bottom left corner of the Notification to display only the icon
5. Collapse and alpha out to display nothing

## API

requirements

### Properties

| name        | type             | required | default   | description                                                                            |
| ----------- | ---------------- | -------- | --------- | -------------------------------------------------------------------------------------- |
| actionArea  | string \| array  | false    | false     | text to be displayed as action area                                                    |
| description | string \| array  | false    | undefined | text to be displayed as description                                                    |
| entered     | boolean          | false    | false     | if true, enter animation will be triggered immediately                                 |
| icon        | string           | true     | undefined | URL for the icon asset. If both icon and logo are provided, only the logo will render. |
| logo        | string           | true     | undefined | URL for the icon asset. If both icon and logo are provided, only the logo will render. |
| title       | string \| string | true     | undefined | text to be displayed as title                                                          |

### Methods

#### enter(): null

Triggers enter animation. Pops icon in and the slides notification text out to the left. If an action area is present, it will slide out from the bottom of the notification after the initial animation completes.

#### dismiss(): null

Triggers dismiss animation. Slides notification back to the right and alpha's all of the elements on the Notification out.
