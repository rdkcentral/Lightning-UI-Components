# Wave

This component takes the wave icon and animates each line individually.

## Source

https://github.comcast.com/Lightning/lightning-ui/packages/@lightning/ui-core/src/components/Wave/Wave.js

## Usage

`Wave` can be used without passing in any additional values and simply toggling the animation via stop and start.

```js
import { Wave } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Wave: {
        type: Wave
      }
    };
  }
}
```

### Sizing

`Wave` height dynamically adjust over the course of the animation. You can set the 'initial' Wave height via this.h.

## API

### Style Properties

| name               | type             | description                                                          |
| ------------------ | ---------------- | -------------------------------------------------------------------- |
| duration           | number           | how long the animation should last                                   |
| keyframes          | array of numbers | sets the animation stops for each line                               |
| lockedRatio        | number           | ratio to maintain for line height when adjusting the width           |
| lockedPaddingRatio | number           | ratio to maintain for spacing between lines when adjusting the width |
| radius             | number           | radius to apply to individual lines                                  |
| repeat             | number           | how many times to repeat the line animations (-1 for infinite)       |

### Methods

#### startAnimation(): void

Start the animation of the circles.

#### stopAnimation(): void

Stops the circles animation. When in the stopped state, the component will fallback to all three circles stacked on top of each other.
