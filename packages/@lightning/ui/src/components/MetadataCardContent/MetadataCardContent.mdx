# MetadataCardContent

A component to format metadata on CardContent. Contains a [Provider](?path=/docs/elements-provider--basic) to format [Icon(s)](?path=/docs/elements-icon--png).

## Source

https://github.comcast.com/Lightning/lightning-ui/blob/develop/packages/@lightning/ui/src/components/MetadataCardContent/index.js

## Usage

```js
import { MetadataCardContent } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      MetadataCardContent: {
        type: MetadataCardContent,
        w: 600,
        h: 250,
        title: 'Title',
        description: 'Description',
        details: 'Details',
        provider: {
          visibleCount: args.visibleCount,
          providers: Array(10).fill(xfinityProviderLogoSquare)
        }
      }
    };
  }
}
```

## API

### Properties

| name        | type                                            | required | default   | description                                                                          |
| ----------- | ----------------------------------------------- | -------- | --------- | ------------------------------------------------------------------------------------ |
| title       | string                                          | false    | undefined | title text                                                                           |
| description | string                                          | false    | undefined | description text directly below title                                                |
| details     | string                                          | false    | undefined | details text at bottom left of component                                             |
| provider    | [Provider](?path=/docs/elementsprovider--basic) | false    | undefined | an object of [Provider](?path=/docs/elements-provider--basic) properties to patch in |

### Style Properties

| name             | type             | description                                                                            |
| ---------------- | ---------------- | -------------------------------------------------------------------------------------- |
| detailsTextStyle | string \| object | text style for details                                                                 |
| providerStyle    | object           | style properties to pass along to the [Provider](?path=/docs/elements-provider--basic) |
