# ListItemPicker

ListItem component with the ability to let a user pick from a list of options.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/packages/@lightningjs/ui/src/components/ListItemPicker/ListItemPicker.js

## Usage

ListItemPicker requires a title and a list of options

```js
import { ListItemPicker } from '@lightningjs/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      ListItemPicker: {
        type: ListItemPicker,
        title: 'Title',
        description: 'Description'
      }
    };
  }
}
```

## API

### Properties

`ListItemPicker` has the same properties as the ones listed in [ListItem](?path=/docs/lightning/ui-listitem--listitem) in addition to a couple listed below:

| name          | type     | default   | description                      |
| ------------- | -------- | --------- | -------------------------------- |
| options       | string[] | undefined | list of selectable options       |
| selectedIndex | number   | undefined | index of current selected option |

### Style Properties

| name                 | type   | description                                                          |
| -------------------- | ------ | -------------------------------------------------------------------- |
| arrowAlphaValue      | number | alpha value of the arrows if value is not equal to the min/max value |
| arrowAlphaValueLimit | number | alpha value of the arrows when value is equal to the min/max value   |
| arrowWidth           | number | width of the arrows                                                  |
| arrowHeight          | object | height of the arrows                                                 |
| arrowColor           | string | color of the arrows                                                  |
| iconLeftSrc          | string | path of the icon to use for the left arrow                           |
| iconRightSrc         | string | path of the icon to use for the right arrow                          |
