# Knob

Knob is a circle component that can be added on to other components

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/Knob/Knob.js

## Usage

```js
import { Knob } from '@lightningjs/ui-components';

class Basic extends lng.Component {
  static _template() {
    return {
      Knob: {
        type: Knob
      }
    };
  }
}
```

## API

### Style Properties

| name        | type   | description       |
| ----------- | ------ | ----------------- |
| circleColor | string | color of the knob |
| radius      | number | border radius     |
