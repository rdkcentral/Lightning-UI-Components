# TitleRow

The `TitleRow` component renders a [Row](/docs/layout-row--basic) component with a title displayed as a [TextBox](/docs/elements-textbox--base) above the Row items.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/TitleRow/TitleRow.js

## Usage

`TitleRow` accepts all [properties supported by Row](/docs/layout-row--basic#properties) as well as an optional `title` property.

```js
import { TitleRow } from '@lightningjs/ui-components';
class Example extends lng.Component {
  static _template() {
    return {
      TitleRow: {
        type: TitleRow,
        title: 'Row Title',
        items: [
          // list of item components
        ]
      }
    };
  }
  _getFocused() {
    return this.tag('TitleRow');
  }
}
```

## API

### Properties

**`TitleRow` is a subclass of the [Row component](/docs/layout-row--basic)**,

It allows setting all
[properties, style properties, and child properties](/docs/layout-row--basic#properties)
supported by `Row`.

| name  | type   | required | default   | description                                          |
| ----- | ------ | -------- | --------- | ---------------------------------------------------- |
| title | string | false    | undefined | The title text to be displayed above the `Row` items |

## Methods

`TitleRow` can access [all public methods supported by the Row component](/docs/layout-row--basic#methods)
