Component shows title and description within a card.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/%40lightningjs/ui-components/src/components/Card/CardTitle.js

## Usage

`CardTitle` can display both a title and a description if a title and description are passed in.
`CardTitle` extends [Card](?path=/docs/components-card--card)

```js
import { CardTitle } from '@lightningjs/ui-components';

class Basic extends lng.Component {
  static _template() {
    return {
      Card: {
        type: CardTitle,
        title: 'CardTitle',
        description: 'Description for the Card Title',
        details: 'Details'
      }
    };
  }
}
```

## API

### Properties

| name        | type             | required | default   | description |
| ----------- | ---------------- | -------- | --------- | ----------- |
| description | string \| object | false    | undefined | description |
| details     | string \| object | false    | undefined | details     |

### Style Properties

| name                      | type             | description                        |
| ------------------------- | ---------------- | ---------------------------------- |
| descriptionTextProperties | string \| object | text style of the description text |
| detailsTextProperties     | string \| object | text style of the details text     |
