# Badge

This component is used to display small keywords of metadata, like video or audio quality (ie HD or CC).

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/Badge/Badge.js

## Usage

The basic component just requires a title:

```js
import { Badge } from '@lightningjs/ui-components';
import circle from '../../assets/images/circle.svg';

class Example extends lng.Component {
  static _template() {
    return {
      Badge: {
        type: Badge,
        title: 'HD'
      }
    };
  }
}
```

Or an icon:

```js
import { Badge } from '@lightningjs/ui-components';

class Example extends lng.Component {
  static _template() {
    return {
      Badge: {
        type: Badge,
        icon: circle,
        iconWidth: 50,
        iconHeight: 50
      }
    };
  }
}
```

A title and an icon can also be used together:

```js
import { Badge } from '@lightningjs/ui-components';
import lightning from '../../assets/images/ic_lightning_white_32.png';

class Example extends lng.Component {
  static _template() {
    return {
      Badge: {
        type: Badge,
        title: 'HD',
        icon: lightning
      }
    };
  }
}
```

## API

### Properties

| name       | type   | required | default   | description                                                    |
| ---------- | ------ | -------- | --------- | -------------------------------------------------------------- |
| icon       | string | false    | undefined | path to image or inline SVG XML                                |
| iconWidth  | number | false    | undefined | width of icon (required for SVG icons)                         |
| iconHeight | number | false    | undefined | height of icon (required for SVG icons)                        |
| iconAlign  | string | false    | undefined | side of the text where icon will appear on (`left` or `right`) |
| title      | string | false    | undefined | text to render                                                 |

### Style Properties

| name            | type             | description                                                          |
| --------------- | ---------------- | -------------------------------------------------------------------- |
| backgroundColor | string           | background style properties such as shader and color                 |
| contentSpacing  | number           | padding to add between the icon and the text                         |
| iconColor       | string           | color to be applied to the icon (only works with pure white PNGs)    |
| offsetY         | number           | offset used to vertically center the text within the container       |
| offsetX         | number           | offset used to horizontally center the text within the container     |
| paddingX        | number           | padding to add on the left and right between the text and background |
| paddingY        | number           | padding to add on the top and bottom between the text and background |
| radius          | number \| array  | shape of component                                                   |
| strokeColor     | string           | color of the Badge border                                            |
| strokeWidth     | number           | width of the Badge border                                            |
| textStyle       | string \| object | text style to apply to the title                                     |

### Events

#### $loadedBadge():void

This event is fired when the contents of the badge have finished loading and the background has been appropriately sized based on the text input.
