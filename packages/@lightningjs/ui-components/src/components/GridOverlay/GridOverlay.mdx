# GridOverlay

This component shows the grid layout, including the column spans and guides for margins and gutters.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/GridOverlay/GridOverlay.js

## Usage

`GridOverlay` just has toggles for showing the columns, margins, gutters, and textbox, which are all visible by default.

```js
import { GridOverlay } from '@lightningjs/ui-components';

class Basic extends lng.Component {
  static _template() {
    return {
      GridOverlay: {
        type: GridOverlay
      }
    };
  }
}
```

## API

### Properties

| name        | type    | required | default | description                                                                                                                                       |
| ----------- | ------- | -------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| showColumns | boolean | false    | true    | when true, displays the column span rectangles                                                                                                    |
| showGutters | boolean | false    | true    | when true, displays the rulers for spacing between columns                                                                                        |
| showMargins | boolean | false    | true    | when true, displays the X and Y margin rulers                                                                                                     |
| showSafe    | boolean | false    | true    | when true, displays the X and Y "safe" zone rulers (this is the area that content should not be rendered in to avoid screen clipping on a device) |
| showText    | boolean | false    | true    | when true, displays the textbox containing the exact numbers associated with the screen, margin, safe, and gutter sizes                           |

### Style Properties

| name              | type             | description                              |
| ----------------- | ---------------- | ---------------------------------------- |
| propertyTextStyle | string \| object | text style to apply to the property text |
| valueTextStyle    | string \| object | text style to apply to the value text    |
