# CardRadio

This component is a variant of the `Card` component that includes multiple lines of text and radio functionality.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/packages/@lightningjs/ui-components/src/components/CardRadio/CardRadio.js

## Usage

`CardRadio` can display a title, subtitle, description and details
`CardRadio` extends [CardTitle](?path=/docs/tiles-cards-cardtitle--card-title)

```js
import { CardRadio } from '@lightningjs/ui-components';

class Basic extends lng.Component {
  static _template() {
    return {
      CardRadio: {
        type: CardRadio,
        title: 'CardRadio Title',
        subtitile: 'CardRadio subtitle',
        description: 'CardRadio description',
        details: 'CardRadio disclaimer'
      }
    };
  }
}
```

## API

### Properties

| name        | type    | readonly | description                                                                      |
| ----------- | ------- | -------- | -------------------------------------------------------------------------------- |
| title       | string  | false    | The text to be displayed in the title section of the card.                       |
| subtitle    | string  | false    | The text to be displayed in the subtitle section of the card.                    |
| description | string  | false    | The text to be displayed in the description section of the card.                 |
| details     | string  | false    | The text to be displayed in the details section of the card.                     |
| checked     | boolean | false    | sets initial `checked` value of Radio. Use `toggle()` to change value after init |

### Style Properties

| name                 | type            | description                             |
| -------------------- | --------------- | --------------------------------------- |
| descriptionTextStyle | string \|object | styles applied to the description text. |
| subtitleTextStyle    | string \|object | styles applied to the subtitle text.    |

### Methods

#### isChecked():boolean

returns the checked value (boolean) for the Radio

#### toggle():boolean

Toggles the `checked` state and returns the new state.
