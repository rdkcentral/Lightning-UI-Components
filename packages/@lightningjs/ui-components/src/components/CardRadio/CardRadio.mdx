# CardRadio

This component is a variant of the `Card` component that includes multiple lines of text and radio functionality.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/packages/@lightningjs/ui-components/src/components/CardRadio/CardRadio.js

## Usage

`CardRadio` can display a title, subtitle, description and details
`CardRadio` extends [CardTitle](?path=/docs/tiles-cards-cardtitle--card-title)

```js
import { CardRadio } from '@lightningjs/ui-components';

class Basic extends lng.Component {
  static _template() {
    return {
      CardRadio: {
        type: CardRadio,
        title: 'Title',
        subtitile: 'subtitle',
        description: 'description',
        details: 'details'
      }
    };
  }
}
```

## API

### Properties

| name     | type   | readonly | description                                                   |
| -------- | ------ | -------- | ------------------------------------------------------------- |
| subtitle | string | false    | The text to be displayed in the subtitle section of the card. |

### Style Properties

| name                 | type            | description                                      |
| -------------------- | --------------- | ------------------------------------------------ |
| alpha                | number          | transparency applied to text and radio component |
| descriptionTextStyle | string \|object | styles applied to the description text.          |
| iconHeight           | number          | height of icon                                   |
| iconWidth            | number          | width of icon                                    |
| subtitleTextStyle    | string \|object | styles applied to the subtitle text.             |
