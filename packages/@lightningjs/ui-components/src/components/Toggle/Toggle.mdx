# Toggle

A checkbox component with toggle slider behavior

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/Toggle/Toggle.js

## Usage

The toggle state is determined by the `checked` property, which defaults to `false`

```js
import { Toggle } from '@lightningjs/ui-components';

class Example extends lng.Component {
  static _template() {
    return {
      Toggle: {
        type: Toggle
      }
    };
  }
}
```

The `checked` state can be programmatically changed with `toggle()`.

```js
this.tag('Toggle').toggle();
```

The `toggle()` method is chainable. Let's say your `Toggle` component has `checked=false`

```js
console.log(this.tag('Toggle').checked); //=> true
```

By default, the `onEnter()` method executes `toggle()`. To add or change functionality, override this method:

```js
this.tag('Toggle').onEnter = toggle => {
  // do stuff
  toggle.toggle();
};
```

## API

### Properties

| name     | type    | readonly | description                                    |
| -------- | ------- | -------- | ---------------------------------------------- |
| checked  | boolean | true     | toggle state                                   |
| disabled | boolean | false    | should the component be unavailable to toggle? |

### Style Properties

| name                   | type          | description                                                                                                                                                               |
| ---------------------- | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| backgroundColor        | string        | color of background when unchecked                                                                                                                                        |
| backgroundColorChecked | string        | color of background when checked                                                                                                                                          |
| knobHeight             | number        | height of Toggle knob                                                                                                                                                     |
| knobPadding            | number        | padding to apply around knob                                                                                                                                              |
| knobRadius             | number        | radius of Toggle knob                                                                                                                                                     |
| knobWidth              | number        | width of Toggle knob                                                                                                                                                      |
| knobColor              | string        | color of knob when unchecked                                                                                                                                              |
| knobColorChecked       | string        | color of knob when checked                                                                                                                                                |
| strokeColor            | string        | color of stroke                                                                                                                                                           |
| strokeRadius           | number\|array | set radius of Toggle, which, if not defined, will be calculated based on the radius of the knob and padding (if no radius is desired, it needs to explicitly be set to 0) |
| strokeWeight           | number        | thickness of stroke                                                                                                                                                       |

### Methods

#### toggle(): void

Toggles `checked` state and updates UI accordingly

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event. Calls `toggle()` by default.
