# Checkbox

A checkbox component that can be toggled between a checked an unchecked state.

## Source

https://github.com/rdkcentral/Lightning-UI-Components/blob/develop/packages/@lightningjs/ui-components/src/components/Checkbox/Checkbox.js

## Usage

Simple component that just needs a type, and, if checked, the `checked` flag.

```js
import { Checkbox } from '@lightningjs/ui-components';

class Example extends lng.Component {
  static _template() {
    return {
      Checkbox: {
        type: Checkbox
      }
    };
  }
}
```

## API

### Properties

| name    | type    | readonly | description  |
| ------- | ------- | -------- | ------------ |
| checked | boolean | true     | toggle state |

### Style Properties

| name                   | type   | description                                 |
| ---------------------- | ------ | ------------------------------------------- |
| backgroundColor        | string | color of Checkbox background when unchecked |
| backgroundColorChecked | string | color of Checkbox background when checked   |
| checkColor             | string | color of check icon                         |
| checkH                 | number | height of check icon                        |
| checkSrc               | string | path the icon to use for the checkmark      |
| checkW                 | number | width of check icon                         |
| radius                 | number | radius of Checkbox                          |
| strokeColor            | string | color of the Checkbox border                |
| strokeWidth            | number | width of Checkbox border                    |

### Methods

#### toggle(): void

Toggles `checked` state and updates UI accordingly. The "Enter" key is listened for and will also trigger `toggle()`.
