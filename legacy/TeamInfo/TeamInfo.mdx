# TeamInfo

Pattern for Data Item that displays team information with an optional logo and team specific background.

```js
import { TeamInfo } from '@lightning/ui/patterns';

TeamInfo: {
  type: TeamInfo,
  team: 'Philadelphia Eagles',
  overlayColor: '4278742891',
  overlayImage: path/to/img,
  items: [
    {
      title: 'Standings',
      subtitle: '2nd in NFC East'
    },
    {
      title: 'Next game',
      subtitle: '11/26 at 4:00p vs WAS'
    }
  ]
}
```

### Properties

name|type|required|default|description
--|--|--|--|--
team|text|true|-|team name for announce text
overlayColor|number|false|-|team color displayed in background
overlayImage|path|false|-|team logo overlayed on background
items|object|true|-|list of text items to display

### Items

name|type|required|default|description
--|--|--|--|--
title|text|true|-|text to display in title
subtitle|text|true|-|text to display in subtitle
