# ProgressBar

Component that shows current progress in a range

## Usage

`ProgressBar` can display `progress` as a perctange of the total `w`:

```js
import { ProgressBar } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ProgressBar: {
        type: ProgressBar,
        w: 410,
        progress: 0.5
      }
    };
  }
};
```

The default progress bar width is 410.

## API

### Properties

| name  | type   | required | default | description            |
| ----- | ------ | -------- | ------- | ---------------------- |
| w | number | true     | -       | width of the bar when passed, `w` overrides 410 default |
progress|number|false|0|percentage of the current progress from 0 to 1

### Style Properties

| name              | type   | description                                       |
| ----------------- | ------ | ------------------------------------------------- |
radius|number|radius of the bar corners, 'radius' overrides default
barColor|number,string|bar background, 'barColor' overrides default, formats(#000000|0x0000000|12345678)
progressColor|number,string|progress fill, 'progressColor' overrides default, formats(#000000|0x0000000|12345678)
animationCurve|string|sets the transition timing function the change in progress animation should use
animationDelay|number|sets the amount of time to wait before animating the progress change
animationDuration|number|sets the amount of time the change in progress should take
