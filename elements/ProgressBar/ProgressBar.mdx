# ProgressBar

Component that shows current progress in a range

## Usage

`ProgressBar` can display `progress` as a perctange of the total `w`:

```js
import { ProgressBar } from '@lightningjs/ui-components';

class Example extends lng.Component {
  static _template() {
    return {
      ProgressBar: {
        type: ProgressBar,
        w: 410,
        progress: 0.5,
        radius: 5,
        barColor: 4284637804,
        progressColor: 4127195135
      }
    };
  }
};
```

<Canvas>
  <Story id="elements-progressbar--basic" />
</Canvas>

The default progress bar width is 410.

## API

### Properties

name|type|required|default|description
--|--|--|--|--
h|number|false|8|height of the bar
w|number|false|410|width of the bar when passed, `w` overrides 410 default
progress|number|false|0|percentage of the current progress from 0 to 1
radius|number|false|4|radius of the bar corners, 'radius' overrides default
barColor|number,string|false|0xffceceda|bar background, 'barColor' overrides default, formats(#000000|0x0000000|12345678)
progressColor|number,string|false|4127195135|progress fill, 'progressColor' overrides default, formats(#000000|0x0000000|12345678)
animationDuration|number|false|0|sets the amount of time the change in progress should take
