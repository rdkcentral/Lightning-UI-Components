# MetadataSmall

Content to be displayed inside Tiles using the inset metadata position in [withMetadata](?path=/docs/patterns-withmetadata--basic). Displays title, info, progress bar, and logo.

## Usage

Wrap a Tile in the `withMetadata` mixin, and pass `MetadataSmall` as the `Metadata` prop:

```js
import { MetadataSmall, Tile } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      TileWithMetadata: {
        type: withMetadata(Tile),
        src: 'asset/path/pets.png',
        w: 410,
        h: 230,
        color: 0xff5d7a74,
        metadataLocation: 'inset',
        Metadata: {
          type: MetadataSmall,
          title: 'Movie Title',
          data: 'A description of the movie',
          logo: 'url/path/logo.png'
        }
      }
    };
  }
}
```

The `id` for the preview comes from the url path generated for the story.

## API

### Properties

| name     | type                                                     | required | default | description                                                          |
| -------- | -------------------------------------------------------- | -------- | ------- | -------------------------------------------------------------------- |
| title    | string                                                   | false    | -       | the first line of text                                               |
| data     | [InlineContent](?path=/docs/layout-inlinecontent--basic) | false    | -       | relevant content data at the bottom                                  |
| progress | number                                                   | false    | 0       | percentage of the current progress from 0 to 1. Hidden if value is 0 |
| logo     | string                                                   | false    | -       | URI of a logo to display at bottom of component                      |
