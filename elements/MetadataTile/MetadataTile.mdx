# MetadataTile

Metadata to be displayed along with `Tiles`.

## Source

https://github.comcast.com/Lightning/lightning-ui/packages/@lightning/ui-core/src/components/MetadataTile/MetadataTile.js

## Usage

MetadataTile can be used with title (text), subtitle (inline content) or description (text), and/or a logo.

```js
import { MetadataTile } from '@lightning/ui';
import lightningbolt from '../../assets/images/ic_lightning_white_32.png';
import circle from '../../assets/images/circle.svg';

class Example extends lng.Component {
  static _template() {
    return {
      MetadataTile: {
        type: MetadataTile,
        w: 400,
        h: 300,
        title: 'Title',
        subtitle: [
          '94%',
          {
            icon: lightningbolt,
            style: { color: getHexColor('00ff00') },
            title: 'Green Lightning Bolt'
          },
          '86%',
          {
            icon: 'http://myriad.merlin.comcast.com/select/logo?entityId=8527084350383982239&width=32&height=&ratio=1x1&trim=false',
            title: 'Rotten Tomatoes rating'
          }
        ],
        logo: circle,
        logoWidth: 32,
        logoHeight: 32
      }
    };
  }
}
```

## API

### Properties

| name         | type           | required             | default | description                                                             |
| ------------ | -------------- | -------------------- | ------- | ----------------------------------------------------------------------- |
| description  | string         | false                | -       | description text to place under title (hides subtitle)                  |
| logo         | string         | false                | -       | logo to display at bottom of component                                  |
| logoHeight   | number         | true (if using logo) | -       | height of logo                                                          |
| logoPosition | string         | false                | 'right' | which side to place logo (`right` or `left`)                            |
| logoTitle    | string         | true (if using logo) | -       | title of logo to use for announcer                                      |
| logoWidth    | number         | true (if using logo) | -       | width of logo                                                           |
| subtitle     | inline content | false                | -       | relevant content data in the middle (hidden if description is provided) |
| title        | string         | false                | -       | first line or headline of the content                                   |

### Styles Properties

| name                      | type   | description                                                     |
| ------------------------- | ------ | --------------------------------------------------------------- |
| descriptionTextProperties | object | text properties for description                                 |
| fadeWidth                 | number | width of fade applied to second line if text encroaches on logo |
| logoHeight                | number | height for logo                                                 |
| logoPadding               | number | spacing between logo and secondLine                             |
| logoWidth                 | number | width for logo                                                  |
| subtitleTextProperties    | object | text properties for subtitle                                    |
| titleTextProperties       | object | text properties for title                                       |
