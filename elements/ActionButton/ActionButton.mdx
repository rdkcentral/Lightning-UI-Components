# ActionButton

General purpose, Xfinity-branded [Button](?path=/docs/button--basic)

## Usage

The basic component just requires a title:

```js
import { ActionButton } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      ActionButton: {
        type: ActionButton,
        title: 'My Button',
        backgroundType: 'stroke'
      }
    };
  }
};
```

An `icon` or `checkbox` can also be included. The expected icon size is 40px by 40px. See [Icon](?path=/docs/icon--basic) for options.

## API

### Properties

name|type|required|default|description
--|--|--|--|--
announce|string|true|label or title|announcement text for screen readers
backgroundType|'stroke'\|'fill'\|'float'\|'ghost'|false|'float'|color of `Button` container.
fixed|boolean|false|false|when true, `w` overrides `minWidth`
h|number|false|72|height
icon|string|false|-|URI for icon or inline svg that may live next to the title. See <a href="?path=/docs/elements-badge--icon">Icon</a>
checkbox|object|false|-|object with `checked` and `spacing` properties to prepend button text with a `Checkbox`. See <a href="?path=/docs/elements-checkbox--basic">Checkbox</a>
checkbox.checked|boolean|false|-|controls whether a `Checkbox` rendered in the button is checked or not
checkbox.spacing|number|false|5|controls the amount of space to the right of the `Checkbox`
minWidth|number|false|150|minimum width of the button
padding|number|false|16|left and right padding value
radius|number|false|8|border radius
title|string|false|-|button text
w|number|false|410|width

### Methods

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event
