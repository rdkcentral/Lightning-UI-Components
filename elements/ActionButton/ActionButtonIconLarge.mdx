# ActionButtonIconLarge

Large [ActionButton](?path=/docs/actionbutton--base) for a single icon.

## Usage

The basic component just requires a icon:

```js
import { ActionButtonIconLarge } from '@lightning/ui';
// or
// import { ActionButtonIconLarge } from '@lightning/ui/components/ActionButton';

class Basic extends lng.Component {
  static _template() {
    return {
      ActionButton: {
        type: ActionButtonIconLarge,
        icon: 'path/to/icon',
        backgroundType: 'stroke'
      }
    };
  }
};
```

The expected icon size is 48px by 48px. See [Icon](?path=/docs/icon--basic) for options.

## API

### Properties

name|type|required|default|description
--|--|--|--|--
announce|string|true|label or title|announcement text for screen readers
backgroundType|'stroke'\|'fill'\|'float'\|'ghost'|false|'float'|color of `Button` container.
fixed|boolean|false|true|when true, `w` overrides `minWidth`
h|number|false|96|height
icon|string|false|-|URI for icon or inline svg that may live next to the title. See <a href="?path=/docs/icon--basic">Icon</a>
padding|number|false|16|left and right padding value
radius|number|false|8|border radius
w|number|false|96|width

### Methods

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event
