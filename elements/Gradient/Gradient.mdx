# Gradient

A semitransparent Gradient

## Usage

Add a Gradient overlay to Tiles or other items

```js
import { Gradient } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Gradient: {
        type: Gradient,
        w: 320,
        h: 180
      }
    };
  }
};
```

Include previews:

<Canvas>
  <Story id="elements-gradient--basic" />
</Canvas>

The `id` for the preview comes from the url path generated for the story.

## API

### Properties

name|type|required|default|description
--|--|--|--|--
h | number | true | - | gradient height
w | number | true | - | gradient width
radius | number | false | 8 | gradient corner radius
gradientColor | color | false | Xfintity Theme palette.colors.grey[] | starting color for the gradient
gradientTop | color | false | transparent | color the gradient fades to
