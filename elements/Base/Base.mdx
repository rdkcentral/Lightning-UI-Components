# Base

This component is used to provide a solid starting point for component development. It includes our default mixins and some class methods.

## Usage

Explain how `Base` works with examples:

```js
import Base from '../../elements/Base';

class MyNewComponent extends Base {
  static _template() {
    return {
      // ...
    };
  }
};
```
