# FocusRing

Fancy ring to focus on Items

## Usage

Add a focus ring to tile or other items to provide a nice highlight
and shadow effect

```js
import { FocusRing } from '@lightning/ui';

FocusRing: {
  type: FocusRing,
  w: this.w,
  h: this.h,
  size: 12
}
```

### Properties

name|type|readonly|default|description
--|--|--|--|--
radius|number|false|Xfinity Theme medium|set radius of corners
color|color|false|Xfintity Theme palette.focusRing.primary|main color of focus ring (.96 alpha)
secondaryColor|color|false|false|Xfintity Theme palette.focusRing.secondary|secondary color of focus ring (.08 alpha)
spacing|number|false|12|sets the amount to increase the focus ring by from the initial component size

### Methods

#### startAnimation(): void

Start the animation of the border

#### stopAnimation(): void

Stops the border animation fallback is a solid border gradient of color/secondaryColor or Xfinity Theme default
