# CardArtwork

Short component description

## Usage

Explain how `CardArtwork` works with examples:

```js
import { CardArtwork } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      CardArtwork: {
        type: CardArtwork
      }
    };
  }
}
```

Include previews:

<Canvas>
  <Story id="component--basic" />
</Canvas>

The `id` for the preview comes from the url path generated for the story.

## API

### Properties

| name        | type    | required | default | description                                                                           |
| ----------- | ------- | -------- | ------- | ------------------------------------------------------------------------------------- |
| src         | string  | true     |         | Which image will be displayed                                                         |
| circleImage | boolean | true     | false   | When true, the shape of the image will be a circle                                    |
| imageSize   | number  | true     |         | The value used to determine position and radius if the shape of the image is a circle |
