# Label

General purpose component to help label content.

## Usage

Labels simply require a background color, a title, and a title color. The Label will automatically adjust based on the rendered width of its text.

```js
import { Label } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Label: {
        type: Label,
        title: 'Label',
        style: {
          backgroundColor: '#f6a50a',
          titleColor: '#ffffff'
        }
      }
    };
  }
}
```

## API

### Properties

| name  | type   | required | default | description            |
| ----- | ------ | -------- | ------- | ---------------------- |
| title | string | true     | -       | text to display in label |

### Style Properties

| name              | type   | description                                       |
| ----------------- | ------ | ------------------------------------------------- |
| backgroundColor   | string | color of the label's background                     |
| titleColor        | string | color of text                                     |
| textStyle         | string | text properties of label                            |
| paddingX          | number | padding between left and right of text and label    |
| paddingY          | number | padding between top and bottom of text and label    |
| radiusTopLeft     | number | determines the shape of the top left's corner     |
| radiusTopRight    | number | determines the shape of the top right's corner    |
| radiusBottomRight | number | determines the shape of the bottom right's corner |
| radiusBottomLeft  | number | determines the shape of the bottom left's corner  |
