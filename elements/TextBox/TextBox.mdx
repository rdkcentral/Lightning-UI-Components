# TextBox

General purpose TextBox component

## Usage

Set a content property with your text and specify a property style as a string. Typeography styles can be found in @lightning/ui/Styles/Fonts.js

```js
import { TextBox } from '@lightning/ui/elements';

TextBox: {
  type: TextBox,
  content: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit.',
  style: 'headline1',
  w: 600,
}
```

### Properties

name|type|required|default|description
--|--|--|--|--
content|string|true|-|Text to be displayed in element
style|string/object|false|body1|Style of text to be displayed. Avalilable values display1, display2, headline1, headline2, headline3, body1, body2, body3, button1, button2, callout1, caption. Alternatively an object can be passed in containing any properties that the Lightning text texture supports.
textColor|hex|false|#ffffff|Desired text color.
textAlign|string|false|left|Avalilable values 'left', 'right', or 'center'.
wordWrap|boolean|false|true|Allows long sentences to be broken into new lines.
verticalAlign|string|false|middle|When lineHeight is larger than fontSize, specifies how text should align vertically inside each line. Avalilable values 'top', 'middle', or 'bottom'.
maxLines|number|false|0|Maximum number of lines to display before truncation.
wordWrapWidth|number|false|0|Width set to start wrapping to next line.
textOverflow|string|false|null|Disabled by default. Truncates long blocks of text to size of wordWrapWidth. Active when wordWrap is set to false.
