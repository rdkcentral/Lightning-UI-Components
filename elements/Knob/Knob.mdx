# Knob

Knob is a circle component that can be added on to other components

## Source

https://github.comcast.com/Lightning/lightning-ui/packages/@lightning/ui-core/src/components/Knob/Knob.js

## Usage

```js
import { Knob } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Knob: {
        type: Knob
      }
    };
  }
}
```

## API

### Style Properties

| name        | type   | description         |
| ----------- | ------ | ------------------- |
| circleColor | string | color of the circle |
| circleSize  | number | size of the circle  |
