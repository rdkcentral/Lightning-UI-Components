# Surface

Surface allows for the background of components such as listItems, cards, and tiles to change dependent on the state of the component.

## Usage

```js
import { Surface } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Surface: {
        type: Surface
      }
    };
  }
}
```

## API

### Style Properties

| name                     | type   | description                                             |
| ------------------------ | ------ | ------------------------------------------------------- |
| backgroundColorDisabled  | string | color of the background when disabled                   |
| backgroundColorFocused   | string | color of the background when focused and not disabled   |
| backgroundColorUnfocused | string | color of the background when unfocused and not disabled |
| radius                   | number | set rounded corner radius                               |
