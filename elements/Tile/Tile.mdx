# Tile

General purpose Tile component

## Usage

Just set a src property to load an image. From there you can set blur, rounded corners and drop shadow.
A [Checkbox](?path=/story/elements-checkbox--basic) can optionally be added to the bottom of the tile, either centered or right aligned.

```js
import { Tile } from '@lightning/ui';

Tile: {
  type: Tile,
  src: 'path/to/image',
  w: 320,
  h: 180,
  blur: 2,
  radius: 16,
  shadow: { w: 320, h: 180, x: 10, y: 10, zIndex: -1,
    texture: lng.Tools.getShadowRect(320, 180, 8, 4),
    color: 0x00ff00
  }
}
```

### Properties

| name            | type    | required | default | description                                    |
| --------------- | ------- | -------- | ------- | ---------------------------------------------- |
| blur            | number  | false    | -       | Blur amount from 0-4                           |
| radius          | number  | false    | -       | set rounded corner radius                      |
| imgRadius       | number  | false    | -       | set rounded corner radius of image             |
| shadow          | obj     | false    | -       | create a drop shadow using a Lightning element |
| src             | string  | false    | -       | url to image                                   |
| fallbackSrc     | string  | false    | -       | url to image to use if the requested src fails |
| gradientColor   | color   | false    | -       | overlay gradient color                         |
| focusGradient   | boolean | false    | false   | show gradient overlay on focus                 |
| persistGradient | boolean | false    | false   | always show gradient overlay                   |
| animate         | boolean | false    | false   | animates focus ring                            |
checkbox|object|false|-|object with `checked` and `placement` properties to display a `Checkbox` at the bottom of the tile. See <a href="?path=/docs/elements-checkbox--basic">Checkbox</a>
checkbox.checked|boolean|false|-|controls whether a `Checkbox` rendered in the Tile is checked or not
checkbox.placement|string['centered', 'right']|false|'right'|controls whethere the `Checkbox` is center or right aligned
