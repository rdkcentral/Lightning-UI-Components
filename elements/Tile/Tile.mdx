# Tile

A flexible solution for displaying artwork and meta information. Used to render entity posters for browsing.

## Usage

Allows objects to be passed in that reflect all properties available in Artwork, Badge, Checkbox, Label, ProgressBar, and Metadata. Additionally, metadataLocation and persistentMetadata are available to configure Metadata placement and visibility.

```js
import { Tile } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Tile: {
        type: Tile
      }
    };
  }
}
```

### Properties

| name               | type    | required | default   | description                                                                                                          |
| ------------------ | ------- | -------- | --------- | -------------------------------------------------------------------------------------------------------------------- |
| artwork            | object  | false    | undefined | Object containing all properties supported in the [Artwork component](?path=/docs/elements-artwork--basic)           |
| badge              | object  | false    | undefined | Object containing all properties supported in the [Badge component](?path=/docs/elements-badge--basic)               |
| checkbox           | object  | false    | undefined | Object containing all properties supported in the [Checkbox component](?path=/docs/elements-checkbox--basic)         |
| label              | object  | false    | undefined | Object containing all properties supported in the [Label component](?path=/docs/elements-label--basic)               |
| metadataLocation   | string  | false    | 'bottom'  | Controls where there metadata is displayed in relation to the Tile. Available values are 'bottom' and 'inset'        |
| persistentMetadata | boolean | false    | false     | Metadata will be shown at all times if set to true, otherwise it will only show when the Tile has focus              |
| progressBar        | object  | false    | undefined | Object containing all properties supported in the [ProgressBar component](?path=/docs/elements-progressbar--basic)   |
| metadata           | object  | false    | undefined | Object containing all properties supported in the [MetadataTile component](?path=/docs/elements-metadatatile--basic) |

### Style Properties

| name              | type     | description                                                                      |
| ----------------- | -------- | -------------------------------------------------------------------------------- |
| animationEntrance | object   | Animation values applied on entrance                                             |
| animationExit     | object   | Animation values applied on exit                                                 |
| artworkStyles     | object   | Overwrite any styles in artwork                                                  |
| badgeStyles       | object   | Overwrite any styles in badge                                                    |
| checkboxStyles    | object   | Overwrite any styles in checkbox                                                 |
| focusRing         | boolean  | Flag used by extension layer to apply FocusRing                                  |
| getFocusScale     | function | Function that returns a number representing the scale of component when focused  |
| getUnfocusScale   | function | Function that returns a number representing the scale of componen when unfocused |
| labelStyles       | object   | Overwrite any styles in label                                                    |
| metadataStyles    | object   | Overwrite any styles in metadata                                                 |
| paddingX          | number   | Padding applied to the inside of the left and right of component                 |
| paddingY          | number   | Padding applied to the inside of the top and bottom of component                 |
| progressBarStyles | object   | Overwrite any styles in progressBar                                              |
| radius            | number   | Outer radius of component                                                        |
| disabledAlpha     | number   | Alpha value of tile when tile is disabled                                        |
