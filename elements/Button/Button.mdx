# Button

Basic component that handles styling and functionality for a button. For Xfinity-branded buttons, check out [ActionButton](?path=/docs/actionbutton--base).

## Usage

```js
import { Button } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Button: {
        type: Button,
        title: 'Button'
      }
    };
  }
};
```

Before the button receives its title, it should be in a loading state.

### Sizing

By default, `Button` widths dynamically adjust to account for `title` and `icon` widths and `padding`, with a minimum width defined by `minWidth`.

If you want a specific `Button` width no matter what the size of content, set the `fixed` property to `true`, which will ignore `minWidth`.

If you want a `Button` that is fully dynamic, set `minWidth to `1`.

The `Button` height is always fixed and defaults to 40px.

### Focus

When focused, the background/foreground colors are inverted by default. This can be overridden with theming.

### Icon

All buttons have the ability to be passed an icon. This will position the icon to the left of the title and center it horizontally and vertically.

## API

### Properties

name|type|readonly|default|description
--|--|--|--|--
announce|string|true|label or title|announcement text for screen readers
fixed|boolean|false|false|when true, `w` overrides `minWidth`
h|number|false|40|height of the button
icon|object|false|-|`src`, `size`, and `spacing` properties for an icon. See <a href="?path=/docs/icon--basic">Icon</a>
label|string|false|-|announcement text to append in front of ", Button" for screen readers
loading|object|false|-|`tag` and `animation` properties for button loading animation
minWidth|number|false|150|minimum width of the button
padding|number|false|10|left and right spacing between content and container edges
radius|rumber|false|0|corner radius for a button
stroke|boolean|false|false|whether to display a border
strokeColor|number|false|0|border color
strokeWeight|number|false|2|thickness of border stroke
title|string|false|-|button text
w|number|false|150|width of the button

### Methods

#### onEnter(thisArg)

Can be overridden to apply handling for the <kbd>Enter</kbd> key
