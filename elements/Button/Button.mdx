# Button

Basic component that handles styling and functionality for a button.

## Usage

Used to convey an interactable portion of the interface, a button can present text, an icon, a checkbox, or both text and an icon or checkbox. It has support for a focused, unfocused, and disabled states.

```js
import { Button } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Button: {
        type: Button,
        title: 'Button'
      }
    };
  }
}
```

### Sizing

By default, `Button` widths dynamically adjust to account for `title` and `icon` widths and `padding`, with a minimum width defined by `minWidth`.

If you want a specific `Button` width no matter what the size of content, set the `fixed` property to `true`. The `minWidth` here will still be honored.

If you want a `Button` that is fully dynamic, set `minWidth` to `1`.

If you'd like to have a fixed button with a width smaller than the `minWidth`, you'll need to set the `minWidth` to a value less than or equal to the provided width.

### Focus

When focused, the background/foreground colors are inverted by default. This can be overridden with theming.

### Prefix

All buttons support two types of Prefixes, a checkbox or an icon.

### Icon

If the button is provided an `icon`, it will be positioned to the left of the title and centered it horizontally and vertically.

### Checkbox

If the button `checkbox` property is set to `true`, a `checkbox` will be positioned to the left of the title and centered it horizontally and vertically.

## API

### Properties

| name     | type    | readonly | default  | description                                                                                |
| -------- | ------- | -------- | -------- | ------------------------------------------------------------------------------------------ |
| checkbox | object  | false    | -        | if defined, a checkbox will be rendered. contains the properties `checked` and `disabled`. |
| disabled | boolean | false    | false    | controls the disabled/enabled state of the button                                          |
| fixed    | boolean | false    | false    | when true, `w` overrides dynamically calclulated width                                     |
| icon     | string  | false    | -        | URI for the icon asset                                                                     |
| justify  | string  | false    | 'center' | alignment of the button's content                                                          |
| title    | string  | false    | -        | button text                                                                                |
| w        | number  | false    | -        | width of the button, if `fixed` is true                                                    |

### Style Properties

| name               | type                          | description                                                                            |
| ------------------ | ----------------------------- | -------------------------------------------------------------------------------------- |
| fontOffsetY        | number                        | used to vertically align the content in the container due to actual font sizing issues |
| justify            | 'center' \| 'left' \| 'right' | alignment of the button's content                                                      |
| minWidth           | number                        | used as a fallback width if no width is passed into the component and `fixed` is false |
| paddingX           | number                        | space between the button horizontal edge and the content                               |
| paddingXIcon       | number                        | space between the button horizontal edge and the content if no title is provided       |
| paddingY           | number                        | space between the button vertical edge and the content                                 |
| prefixMargin       | number                        | the space between the prefix and the button title                                      |
| prefixStyle        | object                        | styles that will be applied to the prefix(icon or checkbox)                            |
| radius             | number                        | shape of component                                                                     |
| textColorFocused   | string                        | color of the title text when the component is focused                                  |
| textColorUnfocused | string                        | color of the title text when the component is not focused                              |
| textColorDisabled  | string                        | color of the title text when the component is disabled                                 |
| textStyle          | string                        | font properties to apply to the title                                                  |
