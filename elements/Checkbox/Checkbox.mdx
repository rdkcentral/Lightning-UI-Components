# Checkbox

A checkbox has 2 states: "on" or "off". Checkboxes can be checked in multiples in a group.

## Usage

Simple component that just needs a type.

```js
import { Checkbox } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Checkbox: {
        type: Checkbox
      }
    };
  }
};
```

## API

### Properties

name|type|readonly|description
--|--|--|--
checked|boolean|true|toggle state

### Methods

#### toggle(): void

Toggles `checked` state and updates UI accordingly
