# Checkbox

A checkbox has 3 states: "on", "off", or "disabled". Checkboxes can be checked in multiples in a group.

## Usage

Simple component that just needs a type.

```js
import { Checkbox } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      Checkbox: {
        type: Checkbox
      }
    };
  }
}
```

## API

### Properties

| name     | type    | readonly | description                                                                                                                                      |
| -------- | ------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| checked  | boolean | true     | toggle state                                                                                                                                     |
| disabled | boolean | true     | disabled the Checkbox, preventing Enter key presses to trigger toggle (Note: does not disable functionality of calling `toggle` method directly) |

### Style Properties

| name                     | type   | description                                 |
| ------------------------ | ------ | ------------------------------------------- |
| radius                   | number | radius of Checkbox                          |
| strokeWidth              | number | width of Checkbox border                    |
| checkW                   | number | width of check icon                         |
| checkH                   | number | height of check icon                        |
| strokeColor              | string | color of the Checkbox border                |
| disabledStrokeColor      | string | color of the Checkbox border when disabled  |
| checkColor               | string | color of check icon                         |
| checkedBackgroundColor   | string | color of Checkbox background when checked   |
| uncheckedBackgroundColor | string | color of Checkbox background when unchecked |
| disabledBackgroundColor  | string | color of Checkbox background when disabled  |

### Methods

#### toggle(): void

Toggles `checked` state and updates UI accordingly. The "Enter" key is listened for and will also trigger `toggle()`.
