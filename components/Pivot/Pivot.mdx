# Pivot Button

Button component used specifically for pivoting to new experiences.

## Usage

The basic component just requires a title:

```js
import { Pivot } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Pivot: {
        type: Pivot,
        title: 'Pivot'
      }
    };
  }
};
```

## API

### Properties

name|type|required|default|description
--|--|--|--|--
w|number|false|185|width
h|number|false|48|height
padding|number|false|16|left and right padding value
title|string|false|-|button text
icon|string|false|-|URI for icon that may live next to the title
backgroundType|'stroke'\|'fill'\|'float'\|'ghost'|false|'float'|color of `Button` container.
