# FocusRing

Fancy ring to focus on Items

## Usage

Add a focus ring to tile or other items to provide a nice highlight
and shadow effect

```js
import { FocusRing } from '@lightning/ui';

FocusRing: {
  type: FocusRing,
  w: this.w,
  h: this.h,
  blur: 4,
  size: 12,
  shadow: {
    padding: 40,
    blur: 3,
    alpha: 0.7
  },
  imageTexture: this._Item.getTexture()
}
```

### Properties

name|type|readonly|description
--|--|--|--
-|-|-|-
blur|number|false|Blur amount from 0-4
size|number|false|set size of ring (even values only)
shadow|obj|false|create a drop shadow
imageTexture|obj|false|Texture image from an image (optional)
