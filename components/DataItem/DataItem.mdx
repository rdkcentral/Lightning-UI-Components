# DataItem

A DataItem is a simple component that patches in an Xfinity styled background and focus state behind any elements or components.

## Usage

DataItems are usually used to display text heavy data or other data elements that can be focused or have a key press capabilities. Some examples for Data Items are (but not limited to): entity descriptions, sports information, music information, and panel widgets.

Data Items must patch in content that can be any arrangement of elements and components. These contents will remain contained inside of the bounds of the data item.

```js
import { DataItem } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      DataItem: {
        type: DataItem,
          h,
          w,
          contents: { ... }
      }
    };
  }
};
```

## API

### Properties

name|type|required|default|description
--|--|--|--|--
h|number|true|-|height of background and all contents inside
w|number|true|-|width of background and all contents inside
margin|number\|object|false|0|spacing between background edges and content start
contents|object|true|-|contents inside of data item that can be composed of elements and component
