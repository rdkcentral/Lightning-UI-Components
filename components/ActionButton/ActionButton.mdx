# ActionButton

General purpose button component

## Usage

The basic component just requires a height, width, and title:

```js
import { ActionButton } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      ActionButton: {
        type: ActionButton,
        h: 72,
        w: 410,
        title: 'My Button'
      }
    };
  }
};
```
