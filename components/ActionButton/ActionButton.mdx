# ActionButton

General purpose button component

## Usage

The basic component just requires a title:

```js
import { ActionButton } from '@lightning/ui';

class Basic extends lng.Component {
  static _template() {
    return {
      ActionButton: {
        type: ActionButton,
        title: 'My Button'
      }
    };
  }
};
```

## API

### Properties

name|type|readonly|description
--|--|--|--
announce|string|true|announcement text for screen readers
h|number|false|height of the button
title|string|false|button text
icon|string|false|URI for icon that may live next to the title
w|number|false|width of the button
