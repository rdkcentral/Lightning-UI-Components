# ActionButtonSmall

Small [ActionButton](?path=/docs/actionbutton--base).

## Usage

The basic component just requires a title:

```js
import { ActionButtonSmall } from '@lightning/ui';
// or
// import { ActionButtonSmall } from '@lightning/ui/components/ActionButton';

class Basic extends lng.Component {
  static _template() {
    return {
      ActionButton: {
        type: ActionButtonSmall,
        title: 'Action Button',
        backgroundType: 'stroke'
      }
    };
  }
};
```

An `icon` can also be included. The expected icon size is 32px by 32px. See [Icon](?path=/docs/icon--basic) for options.

## API

### Properties

name|type|required|default|description
--|--|--|--|--
announce|string|true|label or title|announcement text for screen readers
backgroundType|'stroke'\|'fill'\|'float'\|'ghost'|false|'float'|color of `Button` container.
fixed|boolean|false|false|when true, `w` overrides `minWidth`
h|number|false|64|height
icon|string|false|-|URI for icon or inline svg that may live next to the title. See <a href="?path=/docs/icon--basic">Icon</a>
minWidth|number|false|185|minimum width of the button
padding|number|false|16|left and right padding value
radius|number|false|8|border radius
title|string|false|-|button text
w|number|false|-|width

### Methods

#### onEnter(thisArg): void

Callback to be executed on <kbd>Enter</kbd> key down event
