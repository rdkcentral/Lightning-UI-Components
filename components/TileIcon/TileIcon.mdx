# TileIcon

Tile component with a centerted icon overlay.

## Usage

Set a src property to load an image, and an icon property to set an overlay icon image. From there you can set blur, rounded corners and drop shadow.

```js
import { TileIcon } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      TileIcon: {
        type: TileIcon,
        src: 'path/to/image',
        icon: 'path/to/icon',
        w: 320,
        h: 180
      }
    };
  }
};
```

### Properties

name|type|required|default|description
--|--|--|--|--
blur|number|false|-|Blur amount from 0-4
radius|number|false|-|set rounded corner radius
shadow|obj|false|-|create a drop shadow using a Lightning element
src|string|false|-|url to image
icon|string|false|-|url to overlay icon
iconColor|number|false|0xffffffff|color of icon as an argb hexadecimal number
iconW|number|true for SVGs|-|width of the icon texture, which is required for SVGs (not for PNGs)
iconH|number|true for SVGs|-|height of the icon texture, which is required for SVGs (not for PNGs)
