# Row

Focus management and smooth scrolling for a horizontally-oriented list of items.

## Usage

Explain how the component works. Use an example:

```js
import { Row } from '@lightning/lightning-ui';

class Basic extends lng.Component {
  static _template() {
    return {
      Row: {
        type: Row,
        items: [
          // list of items
        ]
      }
    };
  }
};
```

### Title

A static title can easily be added to a row

```js
import { Row } from '@lightning/lightning-ui';

class Title extends lng.Component {
  static _template() {
    return {
      Row: {
        type: Row,
        title: { text: 'Title Text' }
        items: []
      }
    }
  }
}
```

<Preview>
  <Story id="row--title" />
</Preview>

The `title` property is an <DocsLink id="lng.Text" /> texture; you can customize it like any other text.

```js
{
  title: { fontSize: 20, wordWrap: false, text: 'Hi' }
}
```

You can show and hide the title with the `showTitle` property.

```js
{
  type: Row,
  title: { text: 'Hide Me' },
  showTitle: false,
  items: []
}
```

### Scrolling

By default, `Row` will scroll horizontally to capture items out of view.

<Preview>
  <Story id="row--side-scrolling" />
</Preview>

You can force scrolling on every selection with `alwaysScroll`

```js
{
  type: Row,
  alwaysScroll: true,
  items: []
}
```
