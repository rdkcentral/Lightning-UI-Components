# ListItemSlider

A [`ListItem`](?path=/docs/listitem--basic) with dedicated [`Slider`](?path=/docs/slider--basic) functionality

## Usage

The `ListItemSlider` can display a `title` and `value`:

```js
import { ListItemSlider } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItemSlider: {
        type: ListItemSlider,
        title: 'List Item Slider',
        value: 50
      }
    };
  }
};
```

In an **unfocused** state, `ListItemSlider` only shows the `value` below the `title`.

When **focused**, the slider is visible and left/right keys will change the `value`. Changes to `value` trigger the `onSliderChange` signal.

```js
import { ListItemSlider } from '@lightning/ui';

class Example extends lng.Component {
  static _template() {
    return {
      ListItemSlider: {
        type: ListItemSlider,
        title: 'List Item Slider',
        value: 50,
        signals: {
          onSliderChange: '_handleChange'
        }
      }
    };
  }

  _handleChange(value) {
    console.log(value)
  }
};
```

## API

### Properties

name|type|required|default|description
--|--|--|--|--
min|number|false|0|minimum allowed value
max|number|false|100|maximum allowed value
step|number|false|1|value multiplier
title|string|true|-|label for slider
value|number|false|0|slider value

### Signals

#### onSliderChange

Triggers on changes to `value`. The signal handler function can accept the `value` and `Slider` instance.

**Example**

```js
// in `static _template()`
{
  type: ListItemSlider,
  signals: {
    onSliderChange: '_handler'
  }
}

_handler(value, slider) {
  console.log(value, slider);
}
```
